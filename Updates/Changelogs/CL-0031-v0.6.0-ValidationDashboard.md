# Changelog - IP-0031 Validation Dashboard Visual Redesign

## [0.6.0] - 2026-02-17 - Stage 6: Glow Effects

### Added
- **Effects/GlowEffect.cs** - Layered glow effect for UI elements
  - Multi-layer soft glow using stacked images
  - Configurable size, intensity, and color
  - Optional pulse animation
  - Static AddTo() factory method

- **Effects/AlarmFlashEffect.cs** - Alarm flashing effect
  - Configurable flash frequency and on-duration
  - Alarm/off color states
  - Activate/Deactivate control

- **Shaders/UI/GlowArc.shader** - Custom UI shader
  - Soft glow blending based on alpha
  - UI clipping rect support
  - Standard UI stencil support

### Technical Notes
- GlowEffect uses 3 layered images with decreasing alpha
- AlarmFlashEffect uses Time.unscaledDeltaTime for time-scale independence
- Shader compatible with Unity UI masking/clipping
