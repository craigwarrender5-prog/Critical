# Changelog - IP-0031 Validation Dashboard Visual Redesign

All notable changes for the Validation Dashboard Visual Redesign implementation.

## [0.5.0] - 2026-02-17 - Stage 5: Mini-Trends Panel

### Added
- **Trends/TrendBuffer.cs** - Ring buffer for time-series data
  - Fixed-capacity circular buffer
  - Auto-scaling or fixed min/max range
  - Normalized value access for rendering
  
- **Trends/MiniTrendStrip.cs** - Sparkline-style trend display
  - Extends MaskableGraphic for procedural line rendering
  - Displays 120 data points per strip
  - Label and current value display
  - Color-coded per parameter
  
- **Panels/MiniTrendsPanel.cs** - Always-visible trend panel
  - 8 trend parameters tracked:
    - RCS Pressure (green)
    - PZR Level (amber)
    - T-avg (orange)
    - Subcooling (blue)
    - Charging (cyan)
    - Letdown (orange)
    - SG Pressure (cyan)
    - Net Heat (purple)
  - 1-second sample interval (sim time)
  - AlwaysVisible = true (TabIndex = -1)

### Changed
- Updated `ValidationDashboardBuilder.cs` to create real MiniTrendsPanel

### Technical Notes
- Ring buffer capacity: 120 samples (2 minutes of sim time)
- Line rendering uses procedural mesh generation
- Trends auto-scale with 5% padding on min/max
