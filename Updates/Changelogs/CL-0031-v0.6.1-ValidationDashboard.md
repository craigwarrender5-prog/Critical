# Changelog - IP-0031 Validation Dashboard Visual Redesign

## [0.6.1] - 2026-02-17 - Validator Scene Integration

### Added
- **ValidationDashboardSceneSetup.cs** - Scene integration component
  - Add to Validator.unity scene alongside HeatupValidationVisual
  - Toggle between new uGUI and legacy OnGUI dashboards
  - Auto-finds persistent HeatupSimEngine
  - Disables legacy dashboard when using new system
  - Fallback to legacy if new dashboard build fails

### Usage
1. Open `Validator.unity` scene in Unity Editor
2. Create empty GameObject named "ValidationDashboardRoot"
3. Add `ValidationDashboardSceneSetup` component
4. Set `Use New Dashboard = true`
5. Save scene

The dashboard will now build when entering Validator view (V key).

## Integration with Existing System

The new dashboard integrates with the existing Validator scene system:

| Component | Location | Purpose |
|-----------|----------|---------|
| SceneBridge | MainScene (persistent) | V key loads Validator scene |
| HeatupValidationVisual | Validator scene | Legacy OnGUI dashboard |
| ValidationDashboardSceneSetup | Validator scene | NEW: Builds uGUI dashboard |
| HeatupSimEngine | MainScene (persistent) | Simulation data source |

**Keyboard Controls** (same as legacy):
- `V` - Switch to Validator view
- `1-8`, `Tab`, `Esc` - Return to Operator Screens
- `F1` - Toggle dashboard visibility
- `Ctrl+1-7` - Switch tabs (new dashboard)
- `F5-F9` - Time acceleration
