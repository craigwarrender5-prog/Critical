# Changelog - IP-0031 Validation Dashboard Visual Redesign

All notable changes for the Validation Dashboard Visual Redesign implementation.

## [0.4.0] - 2026-02-17 - Stage 4: Detail Panels

### Added
- **PrimaryLoopPanel.cs** (Tab 1) - Detailed RCS view
  - Loop temperatures: T_avg, T_hot, T_cold arc gauges
  - Saturation temperature and subcooling margin readouts
  - RCS pressure gauge with pressure rate display
  - Individual RCP status indicators (RCP-1 through RCP-4)
  - RCP heat contribution readout
  - Surge flow readout with directional coloring

- **PressurizerPanel.cs** (Tab 2) - PZR control details
  - PZR level gauge with rate display
  - Steam/water space volume distribution
  - PZR pressure gauge with T_sat reference
  - Heater control: master indicator, power gauge, groups readout
  - Spray control: status indicator, flow readout, ΔT display
  - Relief valves: PORV and Safety valve indicators

- **CVCSPanel.cs** (Tab 3) - Chemical/Volume control
  - Charging system: pump indicator, flow gauge, temperature
  - Letdown system: status, flow gauge, isolation indicator
  - Net CVCS flow with bidirectional gauge
  - VCT level gauge with auto-makeup/divert indicators
  - Boron concentration and rate readouts

- **SGRHRPanel.cs** (Tab 4) - Steam Generators and RHR
  - SG primary side: temperature, tube wall temp
  - SG secondary side: pressure gauge, temperature, T_sat
  - Boiling status indicator
  - Heat transfer: gauge, ΔT, UA coefficient
  - RHR system: status, mode, flow gauge
  - RHR temperatures: inlet, outlet, heat removal

- **AlarmsPanel.cs** (Tab 5) - Full alarm annunciator grid
  - 5 alarm category rows (RCS, PZR, CVCS, SG/RHR, System)
  - 8 alarms per row (40 total alarm tiles)
  - AlarmTileExtended with border and pulse animation
  - Acknowledgment capability (future use)

- **ValidationPanel.cs** (Tab 6) - Physics validation metrics
  - Mass conservation: error, rate, gauge, status
  - Energy balance: net heat, input, removal, gauge
  - Heat sources/sinks: RCP, PZR heaters, SG, RHR
  - Timestep metrics: dt, sim speed, iterations
  - Stability indicators: thermal, pressure, level

### Changed
- Updated `ValidationDashboardBuilder.cs` with generic `CreatePanel<T>()` method
- All 7 tabs now use real panel implementations (no more placeholders)

### Technical Notes
- All panels follow three-column layout pattern
- Panels use ArcGauge, BidirectionalGauge, LinearGauge, DigitalReadout, StatusIndicator
- Data updates at 10Hz via OnUpdateData()
- Color coding reflects operational thresholds

## [0.3.0] - 2026-02-17 - Stage 3: Overview Panel

### Added
- **OverviewPanel.cs** - Main at-a-glance dashboard view with 7 sections
- **OverviewSection_GlobalHealth.cs** - Mass/energy conservation metrics
- **OverviewSection_ReactorCore.cs** - Core temperatures (Tavg, Thot, Tcold, ΔT)
- **OverviewSection_RCS.cs** - RCS pressure, subcooling, RCP status
- **OverviewSection_Pressurizer.cs** - PZR level, pressure, heater/spray status
- **OverviewSection_CVCS.cs** - Charging, letdown, net flow, VCT level
- **OverviewSection_SGRHR.cs** - SG secondary pressure, heat transfer, RHR mode
- **OverviewSection_Alarms.cs** - Alarm summary grid with pulsing indicators
- **ParameterRow** - Reusable label+value display component
- **StatusRow** - Reusable label+indicator display component
- **AlarmTile** - Individual annunciator tile with pulse animation

## [0.2.0] - 2026-02-17 - Stage 2: Gauge Components

### Added
- **ArcGauge.cs** - 180° semicircular gauge with animated needle
- **BidirectionalGauge.cs** - 270° center-zero gauge for +/- values
- **LinearGauge.cs** - Horizontal/vertical bar gauge
- **DigitalReadout.cs** - Large numeric display with thresholds
- **StatusIndicator.cs** - Boolean on/off indicator with pulse
- **GaugeAnimator.cs** - Shared animation utilities and easing functions

## [0.1.0] - 2026-02-17 - Stage 1: Foundation Architecture

- **ValidationDashboardTheme.cs** - Centralized colors and style constants

- **ValidationPanelBase.cs** - Abstract base class for panels
