# Changelog - IP-0031 Validation Dashboard Visual Redesign

All notable changes for the Validation Dashboard Visual Redesign implementation.

## [0.3.0] - 2026-02-16 - Stage 3: Overview Panel

### Added
- **OverviewPanel.cs** - Main at-a-glance dashboard view with 7 sections
- **OverviewSection_GlobalHealth.cs** - Mass/energy conservation metrics display
- **OverviewSection_ReactorCore.cs** - Core temperatures (Tavg, Thot, Tcold, ΔT)
- **OverviewSection_RCS.cs** - RCS pressure, subcooling, RCP status
- **OverviewSection_Pressurizer.cs** - PZR level, pressure, heater/spray status
- **OverviewSection_CVCS.cs** - Charging, letdown, net flow, VCT level
- **OverviewSection_SGRHR.cs** - SG secondary pressure, heat transfer, RHR mode
- **OverviewSection_Alarms.cs** - Alarm summary grid with pulsing indicators
- **ParameterRow** - Reusable label+value display component
- **StatusRow** - Reusable label+indicator display component
- **AlarmTile** - Individual annunciator tile with pulse animation

### Changed
- Updated `ValidationDashboardBuilder.cs` to use real OverviewPanel instead of placeholder
- Added `BackgroundSection` color constant to theme

### Technical Notes
- Overview panel uses two-row grid layout (4 sections top, 3 sections bottom)
- All sections inherit from `OverviewSectionBase` abstract class
- Alarm tiles pulse at 2Hz when alarmed using sine wave animation
- Parameter rows support threshold-based color coding
- All data reads directly from HeatupSimEngine public fields

---

## [0.2.0] - 2026-02-16 - Stage 2: Gauge Components

### Added
- **ArcGauge.cs** - 180° semicircular gauge with animated needle
  - Threshold-based color zones (green/amber/red)
  - SmoothDamp needle animation (100ms default)
  - Digital value readout with unit suffix
  - Procedural mesh generation using MaskableGraphic
  
- **BidirectionalGauge.cs** - 270° center-zero gauge
  - Positive values clockwise (blue), negative counter-clockwise (orange)
  - Zero tick mark at 12 o'clock position
  - Ideal for surge flow, net CVCS flow, pressure rate
  
- **LinearGauge.cs** - Horizontal/vertical bar gauge
  - Threshold-based fill coloring
  - Optional tick marks
  - Configurable orientation
  
- **DigitalReadout.cs** - Large numeric display
  - Threshold-based color coding
  - Smooth value and color transitions
  - Optional glow effect support
  
- **StatusIndicator.cs** - Boolean on/off indicator
  - Pill, circle, and square shapes
  - Pulse animation for warning/alarm states
  - Glow effect for active states
  
- **GaugeAnimator.cs** - Shared animation utilities
  - Easing functions (EaseOutQuad, EaseInOutSine, EaseOutBack, etc.)
  - Value smoothing with velocity tracking
  - Color interpolation helpers
  - Pulse/breathing/flash intensity functions
  - Threshold zone detection utilities

### Technical Notes
- All gauges use Unity's MaskableGraphic for procedural mesh generation
- Factory methods allow programmatic creation without prefabs
- Theme colors applied automatically from ValidationDashboardTheme
- Animation uses Time.unscaledDeltaTime for time-scale independence

---

## [0.1.0] - 2026-02-16 - Stage 1: Foundation Architecture

### Added
- **ValidationDashboardController.cs** - Main controller
  - Singleton pattern for global access
  - Dual-timer refresh system (10Hz data, 60Hz visual)
  - Tab navigation (7 tabs: Overview, Primary, Pressurizer, CVCS, SG/RHR, Alarms, Validation)
  - F1 toggle visibility
  - Ctrl+1-7 tab switching
  - F5-F9 and +/- time acceleration controls
  
- **ValidationDashboardTheme.cs** - Centralized style constants
  - Background colors (Dark, Panel, Header, Gauge, Graph, Section)
  - Functional colors (NormalGreen, WarningAmber, AlarmRed, TripMagenta)
  - Text colors (Primary, Secondary, Dark, Disabled)
  - Trace colors (6 distinct for multi-trace plots)
  - Glow colors with transparency
  - Sizing constants (gauge diameters, padding, row heights)
  - Animation constants (smooth times, transition durations)
  - Helper methods for threshold coloring
  
- **ValidationDashboardBuilder.cs** - Runtime UI constructor
  - Creates Canvas with CanvasScaler (1920x1080 reference)
  - Builds header, tab navigation, main content, mini-trends areas
  - Programmatic hierarchy without prefab dependencies
  
- **TabNavigationController.cs** - Tab bar management
  - Tab buttons with hover/active states
  - Animated underline indicator
  - Click and keyboard navigation
  
- **ValidationPanelBase.cs** - Abstract base for all panels
  - IValidationPanel interface implementation
  - CanvasGroup-based fade animation
  - Template methods for derived classes
  - Utility methods (FormatTime, GetStatusColor, etc.)
  
- **HeaderPanel.cs** - Top header bar
  - Plant mode display with color coding (Mode 5=cyan, 4=amber, 3=green)
  - Phase description
  - Simulation time (HH:MM:SS)
  - Wall clock time
  - Time acceleration display
  - Value caching to minimize UI updates
  
- **ValidationDashboardLauncher.cs** - Runtime instantiation helper
  - Attach to any GameObject to enable dashboard
  - Auto-finds HeatupSimEngine
  - DontDestroyOnLoad for persistence
